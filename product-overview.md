# GoCarte Product Overview

## Overview

GoCarte is a lightweight project management platform designed to help teams organize work, coordinate tasks, and monitor project progress. The application provides a centralized workspace where users can create projects, assign tasks, and track work through a shared dashboard system.

GoCarte emphasizes simplicity and fast onboarding, allowing teams to begin organizing work with minimal configuration. The platform supports collaborative workflows by allowing Workroom members to assign tasks, monitor deadlines, and view project status in real time.

---

## Core Concepts

GoCarte organizes work using four primary elements:

### Workroom

A Workroom represents a team or group using GoCarte. Workrooms contain users, projects, tasks, and collaborative resources used by members of that team.

### Project

Projects organize work related to a specific initiative, such as a marketing campaign, product launch, or software update. Each project contains its own tasks, deadlines, and progress indicators.

### Task

Tasks represent individual units of work within a project. Tasks can include descriptions, due dates, assigned team members, and status indicators. Tasks can also be duplicated across multiple projects when the same work applies to more than one initiative.

### Workroom Sandbox

A Workroom Sandbox is a shared collaborative space where team members can co-author project materials, draft plans, and develop task-related content before it is finalized. Sandboxes support real-time updates and allow teams to experiment with project structures or documentation without affecting the primary project workflow.

---

## Key Features

GoCarte provides several core features that support collaborative project management.

### Project Creation and Organization

Users can create projects to organize work around specific goals or initiatives.

### Task Assignment and Tracking

Tasks can be assigned to team members and tracked using status indicators and due dates.

### Deadline Monitoring and Project Progress

Dashboards display task completion status, ongoing work across projects, and highlight upcoming deadlines.

### Workroom Collaboration Tools

Team members can collaborate through shared Workroom Sandboxes with real-time updates and project-level activity tracking.

### Customizable Dashboards

Users can customize dashboard layouts to prioritize relevant information and personalize the interface.

### In-App Messaging and Notifications

GoCarte provides messaging and notification features that alert users to task assignments, updates, and other project activity.
