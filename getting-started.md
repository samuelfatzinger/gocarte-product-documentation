# Getting Started with GoCarte

## Overview

GoCarte is a lightweight project management system that helps teams organize work, track progress, and collaborate on tasks. Users can launch projects, assign tasks, monitor deadlines, and view project status from a centralized dashboard.

This guide walks you through the steps required to begin using GoCarte, including creating your account, launching your first project, and managing tasks.

---

## Before You Begin

GoCarte requires credential verification before you can create an account. Make sure you have the following available:

- a valid email address  
- access to the GoCarte web application  
- permission to create or join a Workroom  

If you have been invited to a Workroom via email, use the invitation link in the email to create your account.

New accounts are typically assigned to a single Workroom, but permissions can grant access to additional Workrooms.

---

## Create Your GoCarte Account

Follow these steps to create your GoCarte account.

1. Go to the GoCarte login page.  
2. Select **Create Account**.  
3. Enter your email address and password.  
4. Create a username (usernames are first initial and last name by default).  
5. Confirm your email address using the verification message sent to your inbox.  
6. Sign in to your new account using your username, email, and password.  

After signing in, you will be directed to your Workroom dashboard.

---

## Launch Your First Project

GoCarte organizes work using a hierarchy of projects, tasks, and subtasks displayed in the **Task Management Container**. A project’s tasks and subtasks can be expanded or hidden.

To launch a project:

1. From the dashboard, select **New Project**.  
2. Enter a name for your project.  
3. Enter a short project description into the project description field.  
4. Select **Launch Project**.

The new project appears in the **Task Management Container** on the left side of the Workroom dashboard.

Selecting the project name in the Task Management Container opens that project page.

---

## Start a New Task

Tasks represent assigned units of work within a specific project. Personal tasks are automatically designated as **Unaffiliated** on your dashboard.

To create a task:

1. Open the project you created by selecting it from the **Task Management Container** or through the dashboard menu.  
2. Select **New Task** from the project menu.  
3. Enter a task title.  
4. Add a short description of the work required.  
5. Select a due date, or leave blank to label the task **Ongoing**.  
6. Select **Create Task**.

The task now appears in the project task list.

Tasks are nested under projects and can be duplicated across multiple projects.

To duplicate a task:

1. Right-click the task name.  
2. Select **Duplicate Task** from the context menu.  
3. Select the project where you want the duplicated task to appear.

---

## Assign a Task

Assigning tasks helps distribute work among team members.

To assign a task:

1. Open the task you created.  
2. Locate the **Team** field to the right of the task description field.  
3. Select a Workroom team member from the list by checking the box next to their name, or select **All**.  
4. Optionally select the **Notify** box to alert assigned team members via internal messaging.  
5. Click **Accept**.

Team members will see the task appear in their task list with a notification icon.

---

## Track Project Progress

GoCarte tracks project progress automatically based on task status.

To update task progress:

1. Open a task from the project task list.  
2. Change the **Status** field to one of the following:

- **To Do** (yellow)  
- **In Progress** (green)  
- **Completed** (blue)  
- **Cancelled** (red)

3. Click **Update** to save the status.

The **Task Management Container** updates to reflect project and task progress.

---

## Next Steps

After creating your first project and tasks, you can continue exploring GoCarte by:

- inviting additional team members to your Workroom  
- customizing the dashboard layout  
- creating additional projects for other initiatives  
- organizing tasks with due dates and status updates  
- integrating GoCarte with external tools using the GoCarte API  

For more information about API integration, refer to the **GoCarte Developer Guide**.
