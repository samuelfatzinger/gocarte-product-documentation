# Overview

GoCarte provides a centralized workspace where teams can create projects, assign tasks, monitor deadlines, and track progress through shared dashboards.

The platform emphasizes simplicity, fast onboarding, and flexible collaboration through features such as **Workrooms**, **Task Management Containers**, and **Workroom Sandboxes**.

This documentation set includes user guides, troubleshooting resources, and developer integration materials.

# Product Concepts

Several core concepts appear throughout the documentation.

### Workroom
A collaborative workspace containing projects, tasks, and team members.

### Project
An organizational container for work related to a specific initiative.

### Task
An individual unit of work associated with a project.

### Workroom Sandbox
A collaborative drafting space where teams can experiment with project structures or documentation before committing changes to active workflows.

### Task Management Container
The interface panel used to navigate projects, tasks, and archived work items.
